# Google Review 自動化發佈系統 (Web 介面)

## 任務目標
為 Ryan 建立一個具備 Web 介面的 Google Review 自動化發佈系統，支援帳號管理、商家設定及評論排程。

## 用戶需求
*   **自動發佈 Google 評論**：能夠使用提供的 Google 帳號列表自動登入並發佈評論。
*   **Web 介面**：提供一個易於使用的網頁介面來管理和操作。
*   **帳號管理**：能夠匯入、管理多個 Google 帳號 (Email, 密碼, 備用碼)。
*   **商家設定**：能夠設定目標商家 (名稱或 Google Maps 連結)。
*   **評論內容管理**：能夠輸入評論範本，並支援隨機切換以避免重複內容。
*   **任務排程**：能夠設定評論發佈的排程 (例如每天自動執行)。
*   **安全性**：確保帳號資訊在傳輸和儲存時是安全的。

## 架構設計 (初步構想)
*   **前端**：React + TypeScript + TailwindCSS (提供直觀的用戶介面)
*   **後端**：Node.js/Python (處理業務邏輯、API 請求、與自動化瀏覽器互動)
*   **資料庫**：SQLite/MySQL (儲存帳號資訊、商家設定、評論內容、任務狀態等)
*   **自動化核心**：Playwright/Selenium (模擬瀏覽器行為，執行 Google 登入和評論發佈)
